# Nuria Kenya — The Honest Bookstore

Nuria is Kenya's premier online bookstore and marketplace, dedicated to bringing African literature, global bestsellers, and educational resources to every doorstep in Kenya.

## 📖 About the Platform

Nuria facilitates a high-fidelity shopping experience with a catalog of over 21,000 products. The platform is designed for speed, reliability, and community engagement.

### Key Features
- **Curated Catalog**: Smart categorization of 21k+ items into specialized genres (Fiction, Business, Religion, etc.).
- **Smart Search**: Performance-tuned searching across tens of thousands of titles.
- **Vendor Ecosystem**: A dedicated portal for authors and vendors to list and manage their titles.
- **Loyalty Program**: Integrated reward system to support frequent readers.
- **High Performance**: Optimized with server-side pagination and a robust Supabase backend.

## 🚀 Development & Tech Stack

This project is built with:
- **Frontend**: React (Vite) + Tailwind CSS + Radix UI
- **Backend**: Supabase (Auth, DB, Storage)
- **Curation Engine**: Python/SQL automated categorization scripts.

### Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Launch Dev Server**:
   ```bash
   npm run dev
   ```

3. **Production Build**:
   ```bash
   npm run build
   ```

## 🛠️ Infrastructure

- **Source Code**: Managed in this repository.
- **Database**: PostgreSQL hosted on Supabase.
- **CI/CD**: Automated testing via Playwright.
